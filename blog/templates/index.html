Short answers to each of the following questions:

  1. What is a client? A client is a piece of computer hardware or software that accesses a service made available by a server.
  2. What is a server? A server is a computer program or a device that provides functionality for other programs or devices, called "clients".
  3. What is an HTTP request? An HTTP request is a request message sent from the client to the server.
  4. What is Django and why do we use it? Django is a Python-based free and open-source web framework, which follows the model-template-view (MTV) architectural pattern. We use it to ease the creation of complex, database-driven websites.
